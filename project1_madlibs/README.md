\# Project 1: Mad Libs (Automate the Boring Stuff)



\## Problem Statement

Create a Mad Libs program that reads a text file containing words like:

ADJECTIVE, NOUN, VERB.



The program should:

1\. Ask the user to enter words for each placeholder

2\. Replace the placeholders with user input

3\. Save the result into a new text file



\## Concepts Used

\- File handling

\- String replacement

\- User input

\- Command Line usage



\## How to Run

```bash

python madlibs.py
